OpenProject - FAQs - IIT Jammu

How to register for an account in the Openproject of IIT Jammu?

  • Go to https://openproject.iitjammu.ac.in/openproject/account/register
  • Fill in the required details.
  • After approval from the administration, your account is ready to use.

I have created an account, but the system says that my account is Not Activated. Why?

Your account needs to be activated from the administrator’s end. After that, you are ready to use your account.

How to create a New Project?

  • There will be a ‘+’ sign on the top-left corner of the homepage.
  • Fill in Necessary details of the project like description, status, public/private and other details.
  • Congrats, You’ve successfully created your first project.

How do you assign a member to a project?

  • The user can not assign itself to any of the projects.
  • Only Admin has the authority to assign users to a specific project.

How to create a task inside the project?

  • Go to Work Packages.
  • Click on “+Create”
  • Choose the type of task (Task, Milestone, Phase etc.) from the dropdown list.
  • Fill in all the necessary fields.

How to assign a Task to a particular member?

  • While Creating a task, we have the option to assign the member also the accountable person to that particular task

What is the difference between different kinds of work packages?

  • Task -> It is an individual task assigned to some member
  • Phase -> It can be considered as the collection of tasks.
  • Milestone -> It’s an essential task after which we can say that we have completed a good amount of work of the phase.

How can a user submit the progress of a task?

  • The user must click on a particular task under the “Assigned to me” option inside the “Work Packages”.
  • Users can update the status/progress or submit the attachment in any task.

What is a Gantt Chart? Where can I find it?

  • Gantt charts in OpenProject are bar charts showing all the tasks constituting a project. The tasks are listed vertically, with the horizontal axis indicating the time. The length of the taskbars is adjusted to the duration of the task.
  • You can find Gantt charts inside the “Work Packages” option.

What are the boards in OpenProject?

  • Agile boards can be for anything you would like to keep track of within your projects: Tasks to be done, Bugs to be fixed, Things to be reviewed, Features to be developed, Risks to be monitored, Ideas to be spread, anything!
  • The boards consist of lists (columns) and cards

What are the modules in OpenProject? How can I add/remove modules for my project?

  • Modules are the different functionalities in a project. e.g. Calendar, Budgets, Github etc.
  • We can Add/Remove modules from a project via Project Settings -> Modules.

How can I delete a Project from OpenProject?

  • Visit Project Settings, you’ll find a Delete button in the top right corner of the page.

How can I maintain different versions of my Project?

  • You can manage your versions by visiting the Project Settings -> Versions page.
  • There you can add a new Version by selecting the Version button from the top right corner.
